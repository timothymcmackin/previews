val mode : [ `Localhost | `Cloud | `Orchestrator | `Host ]
  • Localhost: Agents and the orchestrator are run on the host machine
  • Cloud: The orchestrator is on the host machine. The agents are run onto the cloud
  • `Orchestrator: The orchestrator is run on a VM. This is mode used by the proxy from the orchestrator point of view.
  • `Host: This mode is run by the host machine that initializes the orchestrator running an a VM.
