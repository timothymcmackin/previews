Octez-protocol-alpha-libs: octez protocol alpha libraries

This is a package containing some libraries related to the Tezos alpha protocol.

It contains the following libraries:

  • Tezos_alpha_test_helpers: Protocol testing framework
  • Tezos_baking_alpha: Base library for `tezos-baker/accuser`
  • Tezos_baking_alpha_commands: Protocol-specific commands for baking
  • Tezos_client_alpha: Protocol specific library for `octez-client`
  • Tezos_dal_alpha: Protocol specific library for the Data availability Layer
  • Tezos_layer2_utils_alpha: Protocol specific library for Layer 2 utils
  • Tezos_protocol_plugin_alpha: Protocol plugin
  • Tezos_protocol_plugin_alpha_registerer: Protocol plugin registerer
  • Tezos_smart_rollup_alpha: Protocol specific library of helpers for `tezos-smart-rollup`
  • Tezos_smart_rollup_layer2_alpha: Protocol specific library for `tezos-smart-rollup`
