Module Types.DocSpec

DocSpec module defines primitives for attaching documentation to kaitai specification files.

See the reference implementation.

type refspec =
  1. | TextRef of string
  2. | UrlRef of {
    1. url : string;
    2. text : string;
    }
type t = {
  1. summary : string option;
  2. refs : refspec list;
}
