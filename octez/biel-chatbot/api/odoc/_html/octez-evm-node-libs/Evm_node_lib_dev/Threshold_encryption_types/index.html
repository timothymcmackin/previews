Module Evm_node_lib_dev.Threshold_encryption_types

Ocaml definitions and encoding of the datatypes used for threshold encryption. They mirror the Rust definition from `etherlink/bin_dsn_node/crates/core/src/types.rs`. Compatibility of the encodings between the Ocaml and Rust types is enforced only for (de)serialization in json format.
