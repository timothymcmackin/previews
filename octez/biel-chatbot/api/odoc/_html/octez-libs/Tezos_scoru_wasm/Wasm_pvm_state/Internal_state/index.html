Module Wasm_pvm_state.Internal_state

This module type defines the state for the PVM. For use in lib_scoru_wasm only.

type tick_state =
  1. | Snapshot
  2. | Decode of Tezos_webassembly_interpreter.Decode.decode_kont
  3. | Link of {
    1. ast_module : Tezos_webassembly_interpreter.Ast.module_;
    2. externs : Tezos_webassembly_interpreter.Instance.extern Tezos_webassembly_interpreter.Instance.Vector.t;
    3. imports_offset : int32;
    }
  4. | Init of {
    1. self : Tezos_webassembly_interpreter.Instance.module_key;
    2. ast_module : Tezos_webassembly_interpreter.Ast.module_;
    3. init_kont : Tezos_webassembly_interpreter.Eval.init_kont;
    4. module_reg : Tezos_webassembly_interpreter.Instance.module_reg;
    }
  5. | Eval of {
    1. config : Tezos_webassembly_interpreter.Eval.config;
    2. module_reg : Tezos_webassembly_interpreter.Instance.module_reg;
    }
  6. | Collect
  7. | Stuck of Wasm_pvm_errors.t
  8. | Padding

General state of the PVM

The following describes the general state transitions.

stateDiagram
Collect --> Padding
Snapshot --> Evaluation
state Evaluation {
  Decode --> Link
  Link --> Init
  Init --> Eval
}
Evaluation --> Padding : evaluation succeeded
Padding --> Snapshot : reboot flag is set
Padding --> Collect : reboot flag is not set
Evaluation --> Stuck : something went wrong
