Module Simple_profiler.Base

The Base functor helps to define other backends without having to write the same functions again and again.

Given a way to get and set a state and an output function, the functor will produce a module implementing the DRIVER interface.

Note that the produced module will try to output report every time stamp, inc, mark, span or stop is used.
