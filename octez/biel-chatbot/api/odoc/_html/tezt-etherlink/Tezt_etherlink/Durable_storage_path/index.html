val no_0x : string -> string

no_0x s removes the prefix 0x of s if it exists.

val normalize : string -> string

normalize s calls no_0x and String.lowercase_ascii on s.

val eth_account : string -> path

eth_account addr is the path to the addr account. The address is "normalized", i.e. lowered and removed the prefix 0x if it exists.

storage addr ?key () is the path to the addr storage's code. key can be provided to get the path of a sub-element in the storage.
