type transactions =
  1. | Hash of string list
  2. | Full of Transaction.transaction_object list
  3. | Empty

Blocks can contain either only hashes or full transaction objects. As the empty list case cannot be discriminated between the two types, `Empty` acts as a wildcard for the empty transaction list.
