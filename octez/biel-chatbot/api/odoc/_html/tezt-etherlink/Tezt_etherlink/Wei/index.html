Module Tezt_etherlink.Wei

Helpers for dealing with units of wei. `wei` is the smallest denomination of ether. 1 eth = 1_000_000_000_000_000_000 wei (10^18)

Please note that none of the functions here perform any bounds checks.

type t

A unit of wei

val one : t

val one_eth : t

1 eth, or 10^18 wei

Convert a string to a t.

val of_eth_int : int -> t

Convert an int amount of eth to a wei one. This doesn't perform any bounds checks.

val of_eth_string : string -> t

Convert a string amount of eth to a wei one. This doesn't perform any bounds checks. It assumes the string represents either an integer or a float - both denoting an amount of eth.

val of_gwei_string : string -> t

Convert a string amount of gwei to a wei one. This doesn't perform any bounds checks. It assumes the string represents either an integer or a float - both denoting an amount of gwei.
