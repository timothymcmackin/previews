add dac_plugin page_store buffer message adds a message to buffer. The buffer is serialized to page_store when it is full. Serialization logic is dependent on the encoding scheme.

finalize dac_plugin page_store buffer serializes the buffer to page_store and returns a root hash that represents the final payload. The serialization logic is dependent on the encoding scheme. buffer is emptied after this call.

deserialize_payload dac_plugin page_store hash deserializes a payload from hash using some hash-based encoding scheme. Any payload serialized by add + finalize can be deserialized by this function.
