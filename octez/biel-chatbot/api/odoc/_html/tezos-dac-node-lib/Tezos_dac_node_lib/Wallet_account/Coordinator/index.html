Module Wallet_account.Coordinator

Module containing the definition of wallet and functions operating on wallets exclusive to Coordinator nodes.

type t = {
  1. public_key_hash : Tezos_crypto.Aggregate_signature.public_key_hash;
  2. public_key : Tezos_crypto.Aggregate_signature.public_key;

The type of an account wallet for Coordinator nodes. An account wallet contains the public key hash and a public key for each account wallet. Secret keys URIs are not included in Coordinator account wallets, as Coordinator nodes never sign any data.
