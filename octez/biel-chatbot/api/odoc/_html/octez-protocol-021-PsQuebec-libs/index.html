Octez-protocol-021-PsQuebec-libs: octez protocol 021-PsQuebec libraries

This is a package containing some libraries related to the Tezos 021-PsQuebec protocol.

It contains the following libraries:

  • Tezos_021_PsQuebec_test_helpers: Protocol testing framework
  • Tezos_baking_021_PsQuebec: Base library for `tezos-baker/accuser`
  • Tezos_baking_021_PsQuebec_commands: Protocol-specific commands for baking
  • Tezos_client_021_PsQuebec: Protocol specific library for `octez-client`
  • Tezos_dal_021_PsQuebec: Protocol specific library for the Data availability Layer
  • Tezos_layer2_utils_021_PsQuebec: Protocol specific library for Layer 2 utils
  • Tezos_protocol_plugin_021_PsQuebec: Protocol plugin
  • Tezos_protocol_plugin_021_PsQuebec_registerer: Protocol plugin registerer
  • Tezos_smart_rollup_021_PsQuebec: Protocol specific library of helpers for `tezos-smart-rollup`
  • Tezos_smart_rollup_layer2_021_PsQuebec: Protocol specific library for `tezos-smart-rollup`
