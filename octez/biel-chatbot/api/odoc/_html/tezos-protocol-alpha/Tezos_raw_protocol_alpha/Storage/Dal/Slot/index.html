module LevelHistories : Storage_sigs.Single_data_storage with type t := Raw_context.t and type value = (Dal_slot_repr.History.Pointer_hash.t * Dal_slot_repr.History.t) list

This single entry stores the cells of the DAL skip list constructed during the block under validation. The list is expected to have exactly number_of_slots elements. Its cells ordering is not specified (and not relevant). A cell's binary encoding is bounded (the only part that is evolving in size over time is the number of backpointers, which is bounded by 64).
