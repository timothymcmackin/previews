Using your wallet

Which wallets can I use?​

You can connect any wallet that supports custom EVM networks:

  • Metamask.
  • Coinbase Wallet
  • Trust Wallet
  • Altme wallet
  • Rabby wallet

The easiest way to connect your MetaMask wallet to Etherlink is to go to the faucet and connect as described in Getting testnet tokens. You can also click these buttons:

MainnetTestnet

Connecting wallets manually​

To connect your wallet manually, use the parameters in Network information.
